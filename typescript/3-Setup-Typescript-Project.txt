## Menambah TypeScript ##
- npm install --save-dev typescript
- https://www.npmjs.com/package/typescript


## Setup TypeScript Project ##
- npx tsc --init
- Semua konfigurasi akan dibuat di file tsconfig.json
- Ubah "module" dari "commonjs" menjadi "ES6"
